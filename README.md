# coordinate
PHP 自作
## 概要
古着のコーディネートの情報共有サイトを作成致しました。

## 使い方
自分の服のコーディネートを投稿でき、他ユーザの投稿を一覧で閲覧できます。

アイテムの詳細や服の着こなし方のポイントなども併せて閲覧できるようになっております。

気に入ったコーディネートがあればいいねすることができるようになっております。
